# Technický audit — fast-ride.cz

## Provedeno v této verzi
- responzivní rozložení pro desktop, tablet a telefon;
- kompaktní blok NOVINKA bez změny schváleného hero textu;
- transparentní pozadí původního loga bez změny logotypu;
- canonical URL, robots.txt a sitemap.xml pro HTTPS doménu;
- Open Graph, Twitter Card, Schema.org LocalBusiness/Electrician;
- optimalizace velkých obrázků pro rychlejší načítání;
- rozměry loga v HTML pro omezení posunů layoutu;
- vlastní 404 stránka;
- CNAME a .nojekyll pro GitHub Pages.

## Po nahrání je potřeba jednorázově
1. Ověřit doménu v Google Search Console a odeslat https://fast-ride.cz/sitemap.xml.
2. Přidat web do Seznam Webmaster a odeslat stejnou sitemapu.
3. Založit/ověřit Google Business Profile, pokud firma chce lokální výsledky a mapy.

## Poznámka
Technické SEO připraví web k indexaci, ale samo nezaručuje první pozici na obecné dotazy. Pořadí ovlivňuje konkurence, historie domény, reference, obsah, lokální profil a odkazy z důvěryhodných webů.
